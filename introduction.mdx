## Overview

Дмитрий Белоусов provides a powerful documentation platform where you create, organize, and manage all your project documentation in one centralized space. You build interactive docs using MDX with rich components like Cards, Steps, and Tabs, ensuring your teams stay aligned with up-to-date information. This default space serves as your starting point for any project, supporting everything from API references to user guides.

The platform emphasizes ease of use, letting you focus on content while handling rendering, search, and navigation automatically. Whether you document software, processes, or policies, Дмитрий Белоусов scales with your needs.

## Quick Start

Get up and running in minutes. Follow these steps to create your first documentation page.

<Steps>
  <Step title="Create a New Space" icon="plus">
    Log in to your dashboard at `https://dashboard.example.com` and select "New Documentation Space".
  </Step>
  <Step title="Add Your First Page" icon="file-text">
    Name it `introduction.mdx` and start with YAML frontmatter.
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Preview changes live, then publish. Share the link with your team.
  </Step>
</Steps>

Here is a simple code example to embed a component in your MDX:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch documentation metadata
  const response = await fetch('https://api.example.com/docs/intro', {
    headers: { Authorization: 'Bearer YOUR_API_KEY' }
  });
  const data = await response.json();
  console.log(data.title); // "Introduction to Дмитрий Белоусов"
  ```
  ```python
  # Fetch documentation metadata
  import requests
  response = requests.get(
      'https://api.example.com/docs/intro',
      headers={'Authorization': 'Bearer YOUR_API_KEY'}
  )
  data = response.json()
  print(data['title'])  # "Introduction to Дмитрий Белоусов"
  ```
</CodeGroup>

## Brand Guidelines

Maintain consistency across your documentation by following these guidelines.

Use the primary brand color `#3B82F6` for accents, buttons, and links. Here is an example CSS snippet:

````css
:root {
  --brand-primary: #3B82F6;
  --brand-text: #1F2937;
}

.button {
  background-color: var(--brand-primary);
  color: white;
  border-radius: 6px;
  padding: 12px 24px;
}
````
